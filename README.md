# Homebrew Tap

Custom Homebrew tap for orion-supernova projects.

## Installation

Tap this repository:

```bash
brew tap orion-supernova/tap
```

Then install any available formula:

```bash
brew install term-beam
brew install shell-star
```

## Updating

```bash
brew update
brew upgrade
```

## Uninstall

```bash
brew untap orion-supernova/tap
```
